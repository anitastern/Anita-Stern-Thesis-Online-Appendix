# Factor Investing in Switzerland Under Varying Market Cycles

This repository contains the online appendix for Anita Stern's Bachelor Thesis at the University of St. Gallen (HSG), 2026.

## Contents

- Code-R scripts used for data processing, factor construction, and regression analysis
- Data-input data files (or links to data sources if you can't upload due to licensing)
- Figures-high-resolution versions of all figures in the thesis
- Tables-full regression output tables

## How to reproduce the results

1. Clone this repository: `git clone https://github.com/YOUR-USERNAME/REPO-NAME.git`
2. Open `code/main.R` in RStudio
3. Run the scripts in order
